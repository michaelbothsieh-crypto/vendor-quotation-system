## 1. 專案概述 (Overview)
本文件詳細規劃「外包廠商報價管理系統」的最後一項任務 (Task 6)，包含兩個核心部分：
1. **儀表板首頁 (Dashboard / `/`)**：載入並列出所有歷史報價單，支援快速搜尋、篩選與各種操作功能。
2. **列印預覽與 Print CSS (`/quotations/[id]/print` 與 `print.css`)**：建立專屬的列印預覽版面與列印樣式，使用戶可以透過 `window.print()` 直接將報價單輸出為標準、無多餘 UI 雜質的 HTML / PDF，並包含雙方簽章欄位與詳細的角色工時金額總計。

## 2. 儀表板首頁設計 (Dashboard)

### 2.1 頁面排版
* **頁面路徑**：`src/app/page.tsx`
* **設計風格**：延續現有系統的藍色、靛藍色與紫色漸層，提供 premium 的視覺感受。
* **快速導覽按鈕**：
  * **建立新報價單**：連往 `/quotations/new`。
  * **合作廠商管理**：連往 `/vendors`。
  * **費率設定**：連往 `/settings`。
* **歷史報價單列表**：
  * 使用表格 (Table) 展示報價單。
  * 欄位包含：
    1. **單號** (如 `Q-20260709-001`，使用徽章樣式)
    2. **專案名稱** (Quotation Title)
    3. **合作廠商** (Vendor Name)
    4. **總工時** (TWD 人天，RD/PM/QC/整合工時總和)
    5. **含稅總額** (新台幣元)
    6. **狀態** (草稿 `DRAFT` / 已寄出 `SENT` / 已核准 `APPROVED`)
    7. **建立時間** (格式：`YYYY/MM/DD`)
  * 提供「編輯 (Edit)」按鈕與「預覽/列印 (Preview/Print)」按鈕。

### 2.2 搜尋與篩選邏輯
* 提供搜尋框，可即時過濾：
  * 專案名稱
  * 報價單單號
  * 廠商名稱
* 當無資料時，顯示精美的 Empty State。

## 3. 列印預覽頁面設計 (Print Preview Page)

### 3.1 頁面排版與結構
* **頁面路徑**：`src/app/quotations/[id]/print/page.tsx` (Client Component)
* **版面組成**：
  1. **我方公司名稱與標題**：置中或偏左的大標題「外包專案開發服務報價單」。
  2. **基本資料區** (2 欄式排版)：
     * 左欄：**客戶抬頭** (合作廠商名稱)、**統一編號**、**聯絡人**、**聯絡電話/信箱**。
     * 右欄：**報價單號**、**報價日期**、**狀態**。
  3. **明細表格 (Item Table)**：
     * 將細項依 `QuotationCategory` 進行視覺分組 (Group)。
     * 欄位包含：項次、描述/項目名稱、RD (天)、PM (天)、QC (天)、整合 (天)、備註、小計金額。
     * 每一類別 (Category) 的第一行或大標題標示該分類名稱。
  4. **工時與金額總計區**：
     * 列出各角色的總天數與總額：
       * RD 總天數 x 費率 = 金額
       * PM 總天數 x 費率 = 金額
       * QC 總天數 x 費率 = 金額
       * 整合總天數 x 費率 = 金額
     * 列出金額總計：
       * **未稅總計**
       * **營業稅 (5%)**
       * **含稅總額** (大字加粗)
  5. **簽章欄位區**：
     * 表格或排版預留：
       * **客戶簽認** (公司蓋章 / 負責人簽名 / 日期)
       * **我方簽認** (公司蓋章 / 經辦人簽名 / 日期)
  6. **操作區 (帶 `.no-print` 樣式)**：
     * 頂部或底部常駐一個功能列，提供「返回首頁」、「返回編輯」、「列印報價單 (PDF)」三個按鈕。
     * 點擊「列印報價單 (PDF)」後觸發 `window.print()`。

## 4. 列印專用 CSS 設計 (Print CSS)

* **頁面路徑**：`src/app/print.css`
* **載入方式**：在 `src/app/layout.tsx` 中引入此 css 檔，或直接在 print 頁面 import。
* **樣式規則 (`@media print`)**：
  * **隱藏非列印元素**：將帶有 `.no-print` 的元素 (如導覽列、按鈕、footer) 設為 `display: none !important`。
  * **重設背景與間距**：去除 body 的背景色與陰影，邊距設為 `0`，讓瀏覽器控制列印邊距。
  * **防止截斷**：
    * 對表格行 (`tr`) 與區塊使用 `page-break-inside: avoid` / `break-inside: avoid`，避免單行細項跨頁被砍成兩半。
  * **改善可讀性**：
    * 確保字體為黑色，表格邊框清晰。
    * 將原本的表單輸入框或可編輯狀態轉換為唯讀純文字外觀，去除框線與 focus 樣式。
